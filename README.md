# 🍱 GharConnect  
### Rebuilding Food Systems Through Community, Trust & Technology  

## 🚨 Problem Statement
Millions of urban students and working professionals rely on unhealthy, overpriced restaurant food daily — not by choice, but due to the absence of a structured system connecting them to home-cooked meals nearby.

At the same time, countless homemakers:
- Cook nutritious meals every day  
- Possess valuable culinary skills  
- Yet lack a platform to monetize their abilities  

---

## 💡 Solution: GharConnect

GharConnect is a hyperlocal platform that connects:
- Home chefs (homemakers)  
- Students & professionals (users)  

It enables trust-based, affordable, and healthy meal exchanges within neighborhoods.

---

## ✨ Key Features

### 🔍 Smart Meal Discovery
- Browse nearby home-cooked meals  
- Filter by cuisine, price, dietary needs  
- Real-time availability  

### 🤖 AI-Powered Insights (Core Innovation)
- Suggests upcoming festivals & traditional dishes  
- Recommends cooks what to prepare in advance  
- Notifies users about rare home-style foods  
- Drives demand and supply intelligently  

### 📅 Pre-Order & Scheduling System
- Users can schedule meals in advance  
- Ensures zero wastage and better planning  

### 📍 Hyperlocal Matching
- Works without heavy map dependency  
- Uses pincode and landmark-based clustering  
- Lightweight and cost-efficient  

### 🔐 Secure Order Lifecycle
- Order → confirmation → preparation → delivery/pickup  
- Timeout-based cancellation using custom scheduler  

### 📩 Real-Time Notifications
- SMS alerts for order updates and reminders  

### 🎥 Live Cook Interaction
- Real-time communication between user and cook  
- Builds trust and transparency  

---

## 🧠 Technical Highlights

- Custom Local Scheduler replacing paid cloud tasks  
- Secure real-time token generation  
- RESTful API architecture  
- Modular backend design  
- NoSQL optimized data structure  
- Fault-tolerant system  

---

## 🏗️ Tech Stack

- Frontend: React (Vite)  
- Backend: Node.js + Express  
- Database: Firebase / NoSQL  
- Realtime: Agora RTC  
- Notifications: Twilio SMS  
- Hosting: Vercel / Firebase  

---

## 📊 System Overview

- User interacts with platform  
- Backend processes requests and matches with cooks  
- AI engine enhances recommendations  
- Scheduler handles async operations  
- Secure APIs manage communication
